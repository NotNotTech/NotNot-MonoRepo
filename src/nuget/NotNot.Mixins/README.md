# NotNot.Mixins

Source generator for inline composition/mixins pattern - enables composing classes by inlining members from other classes.

## Ported from InlineComposition

This project is a port of [InlineComposition](https://github.com/BlackWhiteYoshi/InlineComposition) by BlackWhiteYoshi (MIT licensed).

**Port/modifications:** Copyright (c) 2025 NotNot Project and Contributors

## Current Status

Initial port from InlineComposition v1.5.0. Future versions will add support for cross-assembly inlining (currently limited to same-project source).

## Usage

See original [InlineComposition documentation](https://github.com/BlackWhiteYoshi/InlineComposition) for usage patterns.
